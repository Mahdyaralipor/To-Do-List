# To Do List

This is a simple To Do List

## Run Locally

Clone the project

```bash
  git clone https://github.com/Mahdyaralipor/To-Do-List.git
```

Go to the project directory

```bash
  cd To-Do-List
```

Install dependencies

```bash
  npm install
```

Install dependencies And Start the server

```bash
  npm run start
```
